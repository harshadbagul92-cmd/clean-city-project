CleanNashik – Garbage Management Portal
CleanNashik is a fully functional, front-end web application built to empower citizens of Nashik to actively participate in keeping their city clean. Developed using pure HTML, CSS, and JavaScript, the portal offers a complete garbage management ecosystem — from reporting to education to community rewards.
🌍 Problem It Solves
Nashik faces serious challenges with unmanaged garbage in public spaces. Citizens often notice garbage but have no easy way to report it or track whether action was taken. CleanNashik bridges this gap by providing a simple, beautiful, and accessible platform for every citizen.
✨ Key Features
The platform includes a Garbage Reporting System with live camera access and GPS auto-detection, allowing citizens to photograph and report garbage in under 30 seconds. An Area Tracker monitors cleanliness status across four key zones — Panchavati, Nashik Road, Dwarka, and College Road — with live pending and resolved complaint counts. An interactive Waste Segregation Guide helps users identify the correct dustbin for any item from a database of 80-plus waste items. An Awareness and Education section covers harmful effects of garbage, types of waste, Do's and Don'ts, and daily eco-tips. A 10-question Eco Quiz awards badges and points to encourage learning. A Community Leaderboard tracks the top eco-citizens by points. A News Tab allows admins to post updates and publicly name individuals caught littering to create social accountability. A secure Admin Portal with password protection lets administrators manage all reports, users, and news content. Every registered user has a personal Profile Page showing their reports, uploaded photos, and earned points. A full Complaint History table records every report with Who, When, What, and Which details.
🛠️ Technology Used

HTML5 — Structure and layout of all 11 pages
CSS3 — Olive green and beige cream nature-inspired design with animations and full mobile responsiveness
JavaScript (ES6) — Camera API, Geolocation API, localStorage database, Canvas charts, quiz logic, points system, admin authentication

🎨 Design Philosophy
The website uses a calming olive green and beige cream colour palette inspired by nature. Smooth animations, floating particles, an orbiting Earth graphic, and card-based layouts create an immersive experience that keeps users engaged and connected to the environmental cause.
